# Public Dotfiles Staging

This directory stages the public, reproducible parts of the WSL development environment for a future public dotfiles repo.

Layout:

- `chezmoi/`
  Staged files using `chezmoi`-style naming.
- `manifests/public-dotfiles-manifest.md`
  Source mapping and curation notes.

Deliberate exclusions:

- private credentials and secrets
- all Khoj-related config, data, and environment variables
- machine-local caches, logs, histories, and runtime state
- autogenerated files that are not useful for restore
- machine-specific launcher and trust state
