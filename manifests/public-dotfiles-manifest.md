# Public Dotfiles Manifest

Date: 2026-06-13

## Included

- `chezmoi/dot_bashrc`
  From `~/.bashrc`, kept with current shell initialization.
- `chezmoi/dot_bash_profile`
  From `~/.bash_profile`.
- `chezmoi/dot_profile`
  From `~/.profile`, but with KHOJ secrets removed.
- `chezmoi/dot_zshrc`
  From `~/.zshrc`.
- `chezmoi/dot_gitconfig`
  Curated from `~/.gitconfig`.
- `chezmoi/dot_cookiecutterrc`
  Curated from `~/.cookiecutterrc`.
- `chezmoi/dot_config/mimeapps.list`
  From `~/.config/mimeapps.list`.
- `chezmoi/dot_config/RavenCheckers/raven.ini`
  From `~/.config/RavenCheckers/raven.ini`.
- `chezmoi/dot_rye/config.toml`
  From `~/.rye/config.toml`.
- `chezmoi/dot_codex/config.toml`
  From `~/.codex/config.toml`, excluding auth, runtime state, and machine-local project trust entries.
- `manifests/Brewfile`
  Curated Homebrew manifest for reinstall.
- `manifests/apt-packages.txt`
  Curated `apt` package list for reinstall.

## Excluded On Purpose

- all Khoj config, data, and environment values
- `.yarnrc`
  Autogenerated; not useful for restore.
- `.storybook/settings.json`
  Local app state; low restore value.
- `.brew-aliases/`
  Directory is currently empty.
- `.config/wslu/`
  Low-value machine-local state; not worth carrying into the public repo.
- `.codex/auth.json`
  Private auth material.
- `.codex/AGENTS.md`
  Empty file.
