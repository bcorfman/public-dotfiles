# Public Dotfiles Manifest

Date: 2026-06-13

## Included

- `.chezmoi.toml.tmpl`
  Prompts for local identity values during `chezmoi init` and defines local-only machine data keys.
- `dot_bashrc.tmpl`
  From `~/.bashrc`, converted to a template so optional local machine data can be injected privately.
- `dot_bash_profile`
  From `~/.bash_profile`.
- `dot_profile.tmpl`
  From `~/.profile`, with Khoj secrets removed and machine-specific paths moved to local chezmoi data.
- `dot_zshrc.tmpl`
  From `~/.zshrc`, with machine-specific paths moved to local chezmoi data.
- `dot_gitconfig.tmpl`
  Curated from `~/.gitconfig`, with identity and credential-helper values sourced from local chezmoi data.
- `dot_cookiecutterrc.tmpl`
  Curated from `~/.cookiecutterrc`, with identity values sourced from local chezmoi data.
- `dot_config/mimeapps.list`
  From `~/.config/mimeapps.list`.
- `dot_config/RavenCheckers/raven.ini`
  From `~/.config/RavenCheckers/raven.ini`.
- `dot_rye/config.toml`
  From `~/.rye/config.toml`.
- `dot_codex/config.toml`
  From `~/.codex/config.toml`, excluding auth, runtime state, and machine-local project trust entries.
- `docs/LOCAL-SECRETS.md`
  Local-only machine-data and secrets workflow for chezmoi.
- `manifests/Brewfile`
  Curated Homebrew manifest for reinstall.
- `manifests/apt-packages.txt`
  Curated `apt` package list for reinstall.
- `scripts/restore-public-dev-env.sh`
  Scripted base-tool restore for `apt`, Homebrew bootstrap, and `brew bundle`.

## Excluded On Purpose

- all Khoj config, data, and environment values
- machine-specific filesystem paths and credential-helper locations
- repo-only docs and manifests are ignored by `.chezmoiignore.tmpl`
- `.yarnrc`
  Autogenerated; not useful for restore.
- `.storybook/settings.json`
  Local app state; low restore value.
- `.brew-aliases/`
  Directory is currently empty.
- `.config/wslu/`
  Low-value machine-local state; not worth carrying into the public repo.
- `.codex/auth.json`
  Private auth material.
- `.codex/AGENTS.md`
  Empty file.
